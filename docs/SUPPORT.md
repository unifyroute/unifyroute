## Community Help

- Open a GitHub Discussion for questions and design ideas.
- Open a GitHub Issue for reproducible bugs and feature requests.

## What To Include In Support Requests

- UnifyRoute version or commit SHA.
- Environment details (OS, Python, Docker, Redis versions).
- Steps to reproduce.
- Relevant logs and error messages.
- Sanitized config snippets where needed.

## Security Issues

Use the private channel in `SECURITY.md`.
